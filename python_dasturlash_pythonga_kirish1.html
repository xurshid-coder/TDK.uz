Ushbu sahifa Python dasturlash tiliga kirish uchun mo‘ljallangan. Sizga kerakli asosiy tushunchalar, birinchi dastur, o‘zgaruvchilar, ma'lumot tiplari, nazorat strukturalari va amaliy maslahatlar taqdim etiladi. Har bir bo‘lim qisqacha, tushunarli va amaliy misollar bilan boyitilgan.

Python nima?

Python — o‘qilishi oson, yuqori darajadagi, ko‘p maqsadli dasturlash tili. U veb-ishlanmalar, ma'lumotlarni tahlil qilish, sun'iy intellekt, automatizatsiya va o‘qitish uchun keng qo‘llaniladi.

Nega Python o‘rganish kerak?

- Soddaligi va kuchli kutubxonalari (numpy, pandas, flask, django, tensorflow).
- Karyera imkoniyatlari keng.
- Kichik loyihadan katta tizimlargacha moslashadi.

O'rnatish va ish muhitini tayyorlash
  • Python o‘rnatish: python.org saytidan so‘nggi barqaror versiyasini yuklab oling yoki anaconda distribyutivini o‘rnating (ma'lumot tahlili uchun qulay).
  • Tekshirish (Terminal): python --version yoki python3 --version.
  • Muhit: Visual Studio Code yoki PyCharm tavsiya etiladi.
  • Paketlarni o‘rnatish: pip install paket_nomi.
Asosiy tushunchalar

1) Birinchi dastur

Oddiy salomlashuv dasturi:

print("Salom, Python!")

2) O'zgaruvchilar va ma'lumot tiplari

Python-da o'zgaruvchilarni aniqlash oddiy: qiymat tayinlanganda tur avtomatik belgilanadi.

x = 5          # int
pi = 3.14       # float
name = "Ali"   # str
is_ok = True    # bool

3) Kirish va chiqish

ism = input("Ismingiz nima? ")
print("Salom,", ism)

4) Nazorat strukturalari

if x > 0:
    print("Musbat")
else:
    print("Musbat emas")

# sikl
for i in range(5):
    print(i)

while shart:
    do_something()

5) Funksiyalar

def yigindi(a, b):
    return a + b

print(yigindi(3,4))  # 7
Kichik amaliy misollar

1) Sonlarni jamlash

sonlar = [1,2,3,4,5]
print(sum(sonlar))

2) Faylga yozish

with open('hello.txt','w') as f:
    f.write('Salom fayl!')
Maslahat: Kichik loyihalar qiling — masalan, hisoblagich, kontakt daftar, yoki oddiy veb-sahifa (Flask bilan) — bu o‘rganishni chuqurlashtiradi.
