O'zgaruvchilar va Tiplar

Python — Asosiy tushunchalar
Asosiy Turlar Mutability Mashqlar

Bu sahifa Python dasturlash tilida o'zgaruvchilar va ma'lumot tiplari (types) haqida chuqur va amaliy ma'lumot beradi. Har bir bo'lim misollar va tez eslatmalar bilan boyitilgan.

O'zgaruvchi (Variable) nima?

O'zgaruvchi — xotirada saqlanadigan qiymatga berilgan nom. Python-da o'zgaruvchilar uchun maxsus kalit so'z kerak emas: nomni yozib, unga qiymat tayinlaysiz.

# o'zgaruvchi yaratish
x = 10
name = "Aziz"
pi = 3.14
Eslatma: Python dinamik turga ega — o'zgaruvchining turi qiymatga qarab avtomatik belgilanadi.
Asosiy ma'lumot turlari

1. int (butun son)

a = 7     # int

2. float (o'nli son)

b = 3.14  # float

3. str (matn / string)

name = "Salom"  # str

4. bool (mantiqiy)

flag = True   # bool

5. list (ro'yxat)

nums = [1,2,3]   # list

6. tuple (o'zgarmas ro'yxat)

pt = (1,2,3)     # tuple

7. dict (lug'at / xarita)

d = {"ism": "Ali", "yosh": 20}  # dict

8. set (to'plam)

s = {1,2,3}     # set
How to check type: 
type(5)      # <class 'int'>
type("hello") # <class 'str'>
Mutability (o'zgarmaslik / o'zgaruvchanlik)

Ba'zi turlar ichidagi qiymatni o'zgartirish mumkin (mutable), boshqalari esa emas (immutable).

  • Mutable: list, dict, set — element qo'shish, o'chirish mumkin.
  • Immutable: int, float, str, tuple — yangi qiymat berish orqali o'zgartiriladi (asl qiymat o'rniga yangi ob'ekt yaratiladi).

Misol — list va tuple:

lst = [1,2,3]
lst.append(4)  # lst hozir [1,2,3,4]

t = (1,2,3)
# t.append(4)  # xato — tuple uchun append yo'q
Diqqat: Immutable ob'ektlarning o'zgarishi aslida yangi ob'ekt yaratishdir. Bu identifikator (id) o'zgarishida ko'rinadi.
Tiplarni konvertatsiyalash

Ba'zida tiplarni o'zgartirish kerak bo'ladi:

int('10')    # 10
float('3.14') # 3.14
str(100)      # '100'
list((1,2,3))  # [1,2,3]
Eslatma: Konvertatsiya har doim muvaffaqiyatli bo'lmasligi mumkin — masalan, int("3.14") xato beradi.
Operatorlar va tiplar bilan ishlash

Matematik va mantiqiy operatorlar tiplarga bog'liq:

3 + 4      # 7 (int)
3 + 4.5    # 7.5 (float)
"a" + "b" # 'ab' (str concatenation)
[1] + [2]  # [1,2] (list concatenation)
Xatolik: turli turlarni arifmetik qo'shib bo'lmaydi — masalan, "2" + 3 xato beradi.
Amaliy misollar

1) Foydalanuvchi yoshini hisoblash

yosh = int(input("Yoshingiz nechida? "))
print("Sizning yosh: ", yosh)

2) Sonlar bilan ishlash

a = "5"
b = 3
c = int(a) + b  # 8
print(c)

3) Lug'atdan qiymat olish

d = {"ism":"Ali","yosh":25}
print(d.get("ism"))  # Ali
Ko'p uchraydigan xatoliklar
  • Indentatsiya xatolari — Python-da bo'sh joylar muhim.
  • O'zgaruvchi nomini kalit so'z bilan bir xil qilish (masalan, list) — tavsiya etilmaydi.
  • Tipni xato taxmin qilish — input() dan olingan qiymat string bo'ladi.
Mashqlar
  1. Foydalanuvchidan ikki son olib ularning ko'paytmasini chiqaring (tip konvertatsiyasini qo'llang).
  2. Ro'yxatdan faqat musbat sonlarni olib yangi ro'yxat hosil qiling.
  3. Foydalanuvchidan yosh so'rab, agar yosh >= 18 bo'lsa, 'Kattalar' deb chiqaring, aks holda 'Bolalar'.
Tez ko'rsatma
  • Tipni tekshirish: type(x)
  • Tipni o'zgartirish: int(), float(), str(), list()
  • String formatlash: f"Salom, {name}"
