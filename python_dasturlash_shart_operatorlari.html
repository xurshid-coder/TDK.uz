Shart Operatorlari (if-else)

Python — Qarorlar, tekshiruvlar va oqimni boshqarish
Asosiy Sintaksis Operatorlar Misollar

Shart operatorlari dastur oqimini shartga qarab boshqaradi. Ushbu sahifa if, elif, else, shartlarning haqiqiylik qiymati (truthiness), va ko'p uchraydigan xatoliklarni misollar bilan tushuntiradi.

If-else nima?

If-else orqali dastur ma'lum shartlar bajarilganda turli kod bloklarini ishga tushiradi. Bu qaror qabul qilish imkonini beradi.

Misol (inson yoshiga qarab xabar): 
age = 18
if age >= 18:
    print("Siz kattasiz")
else:
    print("Siz bolalarsiz")
Sintaksis va asosiy konstruktsiyalar

1) If

if shart:
    # shart True bo'lsa bajariladigan kod

2) If-else

if shart:
    # agar True
else:
    # aks holda

3) Elif (else if)

if shart1:
    ...
elif shart2:
    ...
else:
    ...
Eslatma: Python-da bloklar indentatsiya (bo'sh joy) bilan belgilanadi. `elif` iroda ketma-ket shartlarni tekshiradi, birinchi True topilganda keyingilar tekshirilmaydi.
Taqqoslash va mantiqiy operatorlar

Shartlarda odatda quyidagi operatorlar ishlatiladi:

  • == — tengmi
  • != — teng emas
  • >, <, >=, <=
  • and, or, not — mantiqiy operatorlar
if x > 0 and x <= 10:
    print("x 1 dan 10 gacha")

if not is_ready:
    print("Tayyor emas")
Diqqat: `=` qiymat tayinlash uchun, `==` esa solishtirish uchun ishlatiladi — noto'g'ri ishlatsa xato bo'ladi.
Truthiness — shartlar qanday baholanadi?

Python-da har bir obyekt ham True, ham False sifatida baholanadi:

  • Bo'sh qiymatlar False qaytaradi: 0, 0.0, "", [], (), {}
  • Qolgan qiymatlar True hisoblanadi.
if []:
    print("True")
else:
    print("False")  # bosadi: False
Maslahat: oddiy tekshiruvlarda if my_list: dan foydalanib ro'yxat bo'sh emasligini tekshash mumkin.
Amaliy misollar

1) Ballar bo'yicha baholash

ball = int(input("Ballingiz: "))
if ball >= 90:
    print("A")
elif ball >= 75:
    print("B")
elif ball >= 60:
    print("C")
else:
    print("F")

2) Foydalanuvchi login tekshiruvi

user = input("Login: ")
if user == "admin":
    print("Xush kelibsiz, admin")
else:
    print("Kirish rad etildi")

3) Nested if (ichma-ich)

age = int(input("Yoshingiz: "))
if age >= 18:
    if age >= 65:
        print("Pensiya yoshida")
    else:
        print("Ishchi yoshida")
else:
    print("Bolalar guruhida")
Ko'p uchraydigan xatoliklar
  • Indentatsiya bilan xato qilish — blok ichida bo'sh joylar mos bo'lishi kerak.
  • `=` o'rniga `==` ishlatish yoki aksincha.
  • Tipni hisobga olmasdan solishtirish: "5" == 5 — False.
  • Murakkab shartlarda mantiqiy operatorlar tartibini hisoblash (bracketsdan foydalaning).
Mashqlar (amaliy)
  1. Foydalanuvchidan son olib, juft yoki toq ekanligini aniqlang.
  2. 3 ta son berilgan: ularni solishtirib eng kattasini toping.
  3. Foydalanuvchidan parol so'rang; to'g'ri bo'lsa "Kirish muvaffaqiyatli", aks holda "Parol xato" deb chiqaring.
Tez ko'rsatma
  • Tekshiruv: if x: — agar x mavjud va haqiqiy bo'lsa.
  • Bir nechta shart: if a && b emas, balki if a and b.
  • Qavslar: if (a > b) and (c < d): — aniqroq va xavfsizroq.
Quick snippets
# ternary operator
x = 10
natija = "Musbat" if x > 0 else "Manfiy"

# chaining comparisons
if 0 < x <= 100:
    print("0 dan 100 gacha")
